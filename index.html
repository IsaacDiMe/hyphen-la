Liturgical Latin hyphenation resources

This page aims at giving some compiled resources coming from the hyphen-la project. The source, tests, documentation, etc. of the files distributed here can be found on the Github repository.

You can find here latest version of the patterns in the libhyphen format (for InDesign, LibreOffice, etc.), as well as the javascript version of Hypher. Note that you can also use the checkSyllabation.py script in the contrib/ folder of Gregorio to check a gabc sore against the InDesign version of the patterns.
